This project is based on react and flask

## 说明

1. 便于管理，会在这个项目中建立一个 `server` 文件夹，编写后台接口。
2. 后台语言使用 python + flask + mongodb + mySQL
3. 仅供学习

## 项目实现

- [x] 主页
- [x] 读书页
- [ ] 图书标签
- [ ] 详情页

## 项目启动

1. `npm i`

2. `python app.js` or `node app.js`

3. `npm start`
